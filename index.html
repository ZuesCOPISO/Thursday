There are different programming languages that involve web development

HTML

HTML is the language used to organize the content of web pages. It uses tags to define the structure of the page, like headings, paragraphs, images, and links.

CSS

CSS, is in charge of how web pages look. It dictates the appearance of HTML elements like colors, fonts, layout, and spacing

JAVA SCRIPT

JavaScript is responsible for adding interactivity and dynamic behavior to web pages
